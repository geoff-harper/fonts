Sample Text

Velocette is a script font that has a classic car feel. There is only one available style for Velocette. This font is best used at larger sizes and, by extension, in headers.

Harlow Solid is a script that has a 50’s diner feel to it. This font only has one style and is best used at larger size. It’s also not ideal for body text and should be saved for headings.

Hamburger Heaven is a script font that one would see in a diner. It has a 50’s feel. Hamburger Heaven is best suited for headings but can also be used for small descriptions.

Budmo is a script that has a ‘Chicago’ type of feel to it. When reading it, one is reminded of Broadway shows. This font has two styles. The other style is the negative of the main font. Budmo is best used for headings.
